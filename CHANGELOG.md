# Changelog

All notable changes to Signal are documented here.

---

## [0.1.0] — 2025-02-18

### Initial release

- Autonomous agent bootstraps on Conway Cloud VM
- Multi-source scanning: Twitter, on-chain, GitHub, news
- AI relevance scoring via Conway Compute
- x402 USDC micropayments on Base — 0.05 USDC / project / month
- Project subscription API (`POST /api/subscribe`)
- Signal feed API (`GET /api/feed`)
- Agent self-funds Conway infra from subscription revenue
- Conway Domains integration for autonomous domain registration
